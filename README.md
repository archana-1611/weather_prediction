## ✨ Overview

I developed **AeroCast** from the ground up to combine deep atmospheric data with intelligent trend summaries. My core objective was to deliver robust data fetching alongside a custom, striking visual identity tailored to Indian users.

### 🌟 What I Built

*   **Architected a Modern React App**: Bootstrapped with Vite for lightning-fast hot module replacement and optimized production builds.
*   **Integrated Free Weather Data**: Hooked into the Open-Meteo API to fetch comprehensive real-time, hourly, and daily forecasting data without requiring expensive API keys.
*   **Custom Indian Geocoding Search**: Implemented a tailored search algorithm that filters results to only show cities within India, ensuring a highly localized experience.
*   **Designed a Premium Glassmorphic UI**: Created a bespoke design system using pure CSS. It features a dark backdrop with dynamic blur effects and Saffron/Green accents reflecting the Indian tricolor.
*   **Built Interactive Data Visualizations**: Utilized `Recharts` to develop an interactive 24-hour temperature forecast chart complete with custom gradients and tooltips.
*   **Developed an "AI" Prediction Engine**: Engineered a logic-based predictive summary card that parses data payloads to generate human-readable insights (e.g., detecting temperature drops and highlighting rain alerts).

## 💻 Getting Started

Follow these instructions to run the project locally on your machine.

### Prerequisites

*   Node.js (v16.x or later)
*   npm (v7.x or later)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/archana-1611/weather_prediction.git
   cd weather_prediction
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the app**:
   Open your browser and navigate to `http://localhost:5173/` or `http://localhost:5174/`.
